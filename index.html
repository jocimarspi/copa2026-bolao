❓ Dúvidas Frequentes
Posso fazer palpite depois do jogo começar? +
Não. O sistema fecha os palpites automaticamente 5 minutos antes do início de cada jogo.
Posso mudar meu palpite depois de salvar? +
Sim! Você pode alterar seu palpite quantas vezes quiser antes dos 5 minutos do início do jogo.
Posso palpitar em todos os jogos de uma vez? +
Sim! Na aba 🎯 Palpites aparecem todos os jogos com prazo ainda aberto.
Como funciona a pontuação? +
5 pontos — Placar exato 🎯

3 pontos — Acertou o resultado (vitória, empate ou derrota)

0 pontos — Errou o resultado
Quem pode participar do bolão? +
O bolão é exclusivo por convite. Somente e-mails cadastrados pelo administrador podem criar conta.
Esqueci minha senha. O que faço? +
Na tela de login, clique em "Esqueci minha senha". O sistema envia um link de recuperação para o seu e-mail.
Meus dados estão seguros? +
Sim. Os dados ficam no Firebase (Google). Nenhum dado sensível financeiro ou pessoal além do nome e e-mail é coletado.
O admin pode ver minha senha? +
Não. Senhas são gerenciadas 100% pelo Firebase Authentication (Google) com criptografia irreversível.
