# Skill: Revisão de Segurança e Vulnerabilidades (Bolão Copa 2026)

Esta skill define critérios de segurança aplicados ao contexto real do projeto Bolão Copa 2026 (HTML/JS + Firebase/Firestore), devendo ser executada sempre que código sensível ou regras de banco forem alteradas.

---

1. **Checklist de Segurança:**
   - Consulte o checklist detalhado para Web e Firebase em [security-checklist.md](resources/security-checklist.md).

## Como usar esta skill
Peça uma revisão de segurança com:
*"Revise o código alterado com base na skill de segurança."*

> ⚠️ Esta skill **não substitui** uma auditoria de segurança profissional. Ela cobre as vulnerabilidades mais comuns no contexto de integrações com o Firebase e segurança do lado do cliente.
